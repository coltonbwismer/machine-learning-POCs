# Machine Learning Proofs of Concept (PoCs)

This repository contains a collection of **machine learning proofs of concept** implemented in Python.  
Each notebook focuses on a core ML algorithm and demonstrates both the **theory** and **practical application** of the technique on example datasets.

The goal of this repository is to showcase hands-on understanding of common machine learning methods, model behavior, and evaluation techniques.

---

## üìÇ Included Notebooks

### Supervised Learning
- **Linear_Regression_PoC.ipynb**  
  Linear regression for modeling continuous target variables and interpreting feature relationships.

- **Logistic_Regression_PoC.ipynb**  
  Binary classification using logistic regression, including decision boundaries and evaluation metrics.

- **KNN_PoC.ipynb**  
  K-Nearest Neighbors classification with distance-based reasoning and parameter tuning.

- **Naive_Bayes_PoC.ipynb**  
  Probabilistic classification using Bayes‚Äô theorem with independence assumptions.

- **Decision_Trees_PoC.ipynb**  
  Decision tree classifiers, feature splits, and interpretability.

- **Random_Forests_PoC.ipynb**  
  Ensemble learning using random forests to improve generalization and reduce overfitting.

- **SVM_PoC.ipynb**  
  Support Vector Machines with multiple kernels (linear, polynomial, RBF), including decision boundary visualizations.

---

### Unsupervised Learning
- **K-Means_PoC.ipynb**  
  Centroid-based clustering and analysis of cluster structure.

- **Hierarchical_Clustering_PoC.ipynb**  
  Agglomerative clustering with dendrogram visualization.

- **DBSCAN_PoC.ipynb**  
  Density-based clustering capable of discovering arbitrarily shaped clusters and identifying noise points.

---

### Dimensionality Reduction
- **PCA_PoC.ipynb**  
  Principal Component Analysis for dimensionality reduction and variance explanation.
